剑指 Offer 13. 机器人的运动范围

地上有一个m行n列的方格，从坐标 [0,0] 到坐标 [m-1,n-1] 。一个机器人从坐标 [0, 0] 的格子开始移动，它每次可以向左、右、上、下移动一格（不能移动到方格外），也不能进入行坐标和列坐标的数位之和大于k的格子。例如，当k为18时，机器人能够进入方格 [35, 37] ，因为3+5+3+7=18。但它不能进入方格 [35, 38]，因为3+5+3+8=19。请问该机器人能够到达多少个格子？(我第一次蠢呼呼的想成了能到达最远的格子)

题目描述

示例 1：

输入：m = 2, n = 3, k = 1
输出：3
示例 2：

输入：m = 3, n = 1, k = 0
输出：1
提示：

1 <= n,m <= 100
0 <= k <= 20

结题思路

这道题其实和前面的有点像，不过这次是规定了起始点和终点的条件改变了。

我们的界限涉及到了边界：别越界（小于长宽）+坐标值和小于K值。

其实到这里，才开始写第二个，你肯定会觉得还是难以下手，我也是，不过想起初中那个男人给我说过的，当你下笔的时候你就已经做出来一半了（当然前提是你自己已经思考了一会了。）
